# 🎵 TuneUp

Practice music theory interactively.

TuneUp is a web platform built to make music theory feel playable instead of memorization-based.

Users can solve theory exercises, train intervals, understand chords, improve fretboard knowledge, and eventually build stronger musical intuition through interactive practice.

---

## ✨ Features

### 🎼 Core Theory
- Chord Spelling
- Chord Naming
- Scale Degree Practice
- Chord Families

### 🎧 Ear Training
- Interval Recognition
- Dictation Exercises
- Listening Challenges

### 🎸 Instrument Practice
- Guitar Theory Exercises
- Fretboard Mapping
- Position Translation

### 🎹 Piano (planned)
- Chord Visualization
- Keyboard Exercises

## 🧠 Motivation

I wanted to create something that feels less like studying and more like practicing music.

The goal is to combine:
- Music Theory
- Ear Training
- Instrument Understanding
- Progress Tracking

into one interactive platform.

## 🛠 Installation

Clone repository:

```bash
git clone YOUR_REPO_LINK
```

Install:

```bash
npm install
```

Run:

```bash
npm run dev
```

Open:

```
http://localhost:5173
```
